#Gestion d'albums:
Cette application permet de gerer une liste d'albums en effectuant plusieurs modifications.
Parmis ces fonctionnalités on a la supprimation d'un album, la sauvegarde, l'ajout de morceau, la lecture de morceaux...

#Ce projet est effectué dans le cadre du cours de conception orientée objet en première année de DUT informatique de l'UCA.


#La partie documentation est dans le sous-dossier Documentation de ListAlbum

#La solution se trouve dans le dossier PROUGERIE_HSOEUR.ListeAlbum.
Le programme doit s'executer une première fois en administrateur depuis l'executable du dossier. Il peut être ensuite lancé depuis le raccourci.
Tous les formats audio de la lecture fonctionnent, sauf le flac dont le temps qui ne s'affiche pas.
Le fichier audio pour Karma Police est fourni pour pouvoir tester le lecteur (mais ne fonctionne pas pour l'installation).
